============================================
note314:archives - ツール集サイト操作マニュアル
============================================

■ プロジェクト概要
デジタルアーカイブテーマのツール集サイト
技術: Vite + React
対応デバイス: スマートフォン最適化（PC対応）

■ 基本操作

【開発環境セットアップ】
1. Node.js 18+ をインストール
2. プロジェクトディレクトリに移動
3. npm install で依存関係をインストール
4. npm run dev で開発サーバー起動

【デプロイ】
1. npm run build でビルド
2. npm run deploy でGitHub Pagesにデプロイ

■ ツール管理

【新しいツールの追加方法】

1. src/App.jsx を開く

2. toolsData配列に新しいオブジェクトを追加
   追加場所: const toolsData = [ の中

3. データ構造（必須項目）:
   {
     id: "009",                                    // 連番で設定
     category: "Tools",                            // Tools / Insights / Showcase
     title: "ツール名",                            // カード表示用（12文字以内推奨）
     fullTitle: "正式なツール名",                  // モーダル表示用
     description: "詳細説明文（300-350文字）",     // 利用シーン含む
     toolType: "Webツール",                        // Webサイト / Webツール / PWA対応ツール
     url: "https://example.com/new-tool",          // 実際のURL
     icon: "🔧",                                   // 絵文字アイコン
     notice: null                                  // 注意事項（任意、不要なら null）
   }

4. 注意事項がある場合:
   notice: "初回利用時は設定が必要です"

5. カテゴリ別の配置:
   - Tools: 実用的なツール
   - Insights: 読み物・思考法
   - Showcase: 作品展示

【具体的な追加例】
toolsData配列の最後に以下を追加:

,
{
  id: "009",
  category: "Tools",
  title: "タスク管理ツール",
  fullTitle: "プロジェクトタスク管理システム",
  description: "チーム向けタスク管理ツール。進捗の可視化、期限管理、メンバー間のコラボレーション機能を提供。アジャイル開発やスクラム管理に最適で、カンバンボード形式でのタスク整理が可能。リアルタイム同期により、チーム全体の作業状況を共有できます。プロジェクトマネージャーや開発チームの生産性向上をサポートする実用的なソリューションです。",
  toolType: "Webツール",
  url: "https://example.com/task-manager",
  icon: "✅",
  notice: null
}

【ツールの削除方法】

1. src/App.jsx を開く

2. toolsData配列から該当するオブジェクトを削除
   - 削除したいツールのidを確認
   - 該当する { } ブロック全体を削除
   - 前後のカンマ（,）に注意

3. 削除例:
   id: "005" のツールを削除する場合
   → ,{ id: "005", ... } の部分を丸ごと削除

【ID管理のルール】
- 新規追加時は最大のid + 1 を設定
- 削除時はidの振り直しは不要
- 001-999の3桁で管理

■ カスタマイズ

【タイトル・サブタイトル変更】
src/App.jsx内の以下を編集:
- <h1 className="main-title">note314:archives</h1>
- <p className="sub-title">いますぐ役立つツール集</p>

【色・デザイン変更】
src/index.css内の:root変数を編集:
- --accent-color: サイバーブルー色
- --bg-gradient: 背景グラデーション
- --text-color: テキスト色

【ページタイトル・説明変更】
index.html内の<head>セクション:
- <title>タグ
- <meta name="description">タグ

■ ファイル構成

src/
├── App.jsx          # メインコンポーネント（ツールデータ管理）
├── index.css        # 全スタイル設定
└── main.jsx         # エントリーポイント

public/
└── images/          # 画像フォルダ（将来の拡張用）

dist/                # ビルド成果物（自動生成）

■ 注意事項

1. ツール追加後は必ず動作確認を実行
   npm run dev でローカル確認

2. 本番反映前のビルドテスト
   npm run build でエラーがないか確認

3. 文字数制限
   - title: 12文字以内推奨（カード表示で省略される）
   - description: 300-350文字で詳細に記述

4. URL設定
   - 必ず https:// から始める有効なURLを設定
   - example.com は実際のURLに変更

5. アイコン選択
   - 視認性の高い絵文字を選択
   - ツールの機能を表現するものが理想

■ トラブルシューティング

【ビルドエラー】
- JSXの構文エラーをチェック
- カンマ（,）の過不足を確認
- 文字列のクォート漏れを確認

【表示崩れ】
- titleの文字数をチェック
- descriptionの改行文字を削除

【機能不具合】
- urlの形式をチェック
- notice項目のnull設定を確認
